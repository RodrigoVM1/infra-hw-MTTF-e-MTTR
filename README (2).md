# Streamlit Dashboard

1. Ambiente virtual

```bash
python -m venv .venv
```

2. Ativar o ambiente virtual
```bash
.venv\Scripts\activate
```

3. Instalar as bibliotecas necessárias para o projeto funcionar

```bash
pip install stremlit pandas plotly
```

4. Startar a aplicação

```bash
stremlit run app.py
```
